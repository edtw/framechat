# framechat

A modern, lightweight chat framework.

## Features

- Clean, minimal architecture
- Real-time messaging support
- Extensible plugin system
- Easy to self-host

## Getting Started

```bash
# Clone the repository
git clone https://github.com/edtw/framechat.git
cd framechat

# Install dependencies
npm install

# Start development server
npm run dev
```

## Project Structure

```
framechat/
├── backend/          # API and real-time services
├── frontend/         # Web client
├── docs/             # Documentation
└── README.md
```
